class plugin.sdk.adapter.base.AdapterBase[源代码]¶

基类：ABC

Adapter 基类

所有 Adapter 插件必须继承此类。

示例:

```python from plugin.sdk.adapter import AdapterBase, AdapterConfig, AdapterContext

class MyAdapter(AdapterBase):
async def on_startup(self) -> None:

self.logger.info("MyAdapter started")

async def on_shutdown(self) -> None:

self.logger.info("MyAdapter stopped")

```
