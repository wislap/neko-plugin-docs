sort(*, by=None, key=None, cast=None, reverse=False)[源代码]¶

Return a new list sorted by fields or a custom key.

中文: 返回一个按字段或自定义 key 排序后的新 BusList. English: Return a new BusList sorted by fields or a custom key.

参数:

  • by (str | Sequence[str] | None) -- 中文: 按哪些字段排序. 可以是单个字段名或字段名列表. 为空时会尝试默认字段 (timestamp/created_at/time). English: Sort by field name(s). If omitted, tries default fields (timestamp/created_at/time).

  • key (Callable[[TRecord], Any] | None) -- 中文: 自定义排序函数. 与 by 互斥. English: Custom sort key callable. Mutually exclusive with by.

  • cast (str | None) -- 中文: 对字段值进行简单类型转换, 支持 "int"/"float"/"str". English: Optional value casting for field values: "int"/"float"/"str".

  • reverse (bool) -- 中文: 是否倒序. English: Sort descending when True.

返回类型:

BusList[TRecord]

备注

  • sort(key=callable) 无法被 reload() 重放.

  • sort(by=...) 可以被 reload()/watch() 重放.

reload(ctx: BusReplayContext) → BusList[TRecord][源代码]¶
reload(ctx: None = None) → BusList[TRecord]

Replay the recorded plan against live bus data.

中文: 使用可重放 plan 重新从 bus 拉取数据并应用同样的链式操作, 返回最新 BusList. English: Reload from bus by replaying the stored plan and operations.

Requirements:

  • 必须是 replayable plan (通常由 get()/filter()/where_*/sort(by=...)/limit() 组合产生).

  • fast_mode=True 或 plan 缺失时不可用.

参数:

  • ctx (BusReplayContext | None)

  • incremental (bool)

返回类型:

BusList[TRecord]

reload_with(ctx: BusReplayContext, *, inplace: bool = False, incremental: bool = False) → BusList[TRecord][源代码]¶
reload_with(ctx: None = None, *, inplace: bool = False, incremental: bool = False) → BusList[TRecord]

Reload with optional in-place mutation.

中文: reload 的底层实现, 可选择 inplace=True 直接更新当前对象内容. English: Underlying reload implementation; can mutate current instance when inplace=True.

参数:

  • ctx (BusReplayContext | None) -- 中文: 需要提供 ctx.bus.messages/events/lifecycle 等接口. English: Context providing ctx.bus.* clients.

  • inplace (bool) -- 中文: True 时原对象会被更新(保持同一实例), False 时返回新列表. English: When True, mutate this instance; otherwise return a new list.

  • incremental (bool)

抛出:

NonReplayableTraceError -- plan 缺失或含不可重放步骤(如 sort(key=callable), where(predicate)).

返回类型:

BusList[TRecord]

异步版本的 reload_with，使用 asyncio.to_thread 包装同步调用。

  • inplace (bool) -- True 时原对象会被更新，False 时返回新列表

watch(ctx: BusReplayContext, *, bus: str | None = None, debounce_ms: float = 0.0) → BusListWatcher[TRecord][源代码]¶
watch(ctx: None = None, *, bus: str | None = None, debounce_ms: float = 0.0) → BusListWatcher[TRecord]

Create a watcher for this query.

中文: 基于当前可重放 plan 创建 watcher, 用于监听 bus 变化并触发 subscribe 回调. English: Create a watcher based on the replayable plan for change notifications.

参数:

  • ctx (BusReplayContext | None) -- 中文: 需要提供 ctx.bus.* 以及(在插件进程内)必要的 IPC 能力. English: Context providing bus clients and (in plugin process) IPC capability.

  • bus (str | None) -- 中文: 手动指定 bus 类型("messages"/"events"/"lifecycle"). 默认从 plan 自动推断. English: Override bus name; otherwise inferred from the plan.

  • debounce_ms (float) -- 中文: 监听去抖(毫秒). >0 时会合并短时间内多次 bus change, 降低 reload 频率. English: Debounce window in milliseconds. When >0, coalesce bursts of bus changes.

返回类型:

BusListWatcher[TRecord]

备注

  • watcher 需要 replayable plan; fast_mode 或 where(predicate) 这类不可重放会报错.
