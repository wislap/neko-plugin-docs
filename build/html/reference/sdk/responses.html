响应格式 API¶

plugin.sdk.responses.ok(data=None, *, code=ErrorCode.SUCCESS, message='', trace_id=None, time=None, **meta)[源代码]¶

创建成功响应的标准格式

用于插件entry返回成功结果。返回统一的响应格式,包含成功标记、数据和元信息。

参数:

  • data (Any) -- 返回的数据,可以是任何可序列化的类型

  • code (ErrorCode) -- 错误码,默认 SUCCESS (0)

  • message (str) -- 可选的消息说明

  • trace_id (str | None) -- 可选的追踪ID,用于日志关联

  • time (str | None) -- 可选的时间戳,默认使用当前UTC时间

  • **meta (Any) -- 额外的元数据,会放入"meta"字段

返回:

{

"success": True, "code": 0, "data": <data>, "message": <message>, "error": None, "time": <ISO时间>, "trace_id": <trace_id>, "meta": <meta> # 如果提供了meta

}

返回类型:

标准响应字典,格式为

示例

>>> ok(data={"result": 42})
{'success': True, 'code': 0, 'data': {'result': 42}, ...}

>>> ok(data=[1, 2, 3], message="处理完成", custom_field="value")
{'success': True, 'code': 0, 'data': [1, 2, 3], 'message': '处理完成', 'meta': {'custom_field': 'value'}, ...}

plugin.sdk.responses.fail(code, message, *, details=None, retriable=False, trace_id=None, time=None, **meta)[源代码]¶

创建失败响应的标准格式

用于插件entry返回错误结果。返回统一的错误响应格式,包含错误码、错误信息和详情。

参数:

  • code (ErrorCode | str | int) -- 错误码,可以使用ErrorCode枚举、整数或自定义字符串

  • message (str) -- 错误描述信息

  • details (Any) -- 可选的错误详情,可以是任何可序列化的类型

  • retriable (bool) -- 是否可重试,True表示客户端可以重试此操作

  • trace_id (str | None) -- 可选的追踪ID,用于日志关联

  • time (str | None) -- 可选的时间戳,默认使用当前UTC时间

  • **meta (Any) -- 额外的元数据,会放入"meta"字段

返回:

{

"success": False, "code": <int>, "data": None, "message": "", "error": {

"code": <code_name>, "message": <message>, "details": <details>, "retriable": <retriable>

}, "time": <ISO时间>, "trace_id": <trace_id>, "meta": <meta> # 如果提供了meta

}

返回类型:

标准错误响应字典,格式为

示例

>>> from plugin.sdk import ErrorCode, fail
>>> fail(ErrorCode.VALIDATION_ERROR, "参数无效")
{'success': False, 'code': 400, 'error': {'code': 'VALIDATION_ERROR', 'message': '参数无效', ...}, ...}

>>> fail("CUSTOM_ERROR", "自定义错误", details={"field": "email"}, retriable=True)
{'success': False, 'code': 500, 'error': {'code': 'CUSTOM_ERROR', 'retriable': True, ...}, ...}

plugin.sdk.responses.is_envelope(value)[源代码]¶

检查值是否是标准响应格式

验证给定的值是否符合ok()/fail()返回的标准响应格式。

参数:

value (Any) -- 要检查的值

返回:

True 如果是标准响应格式,False 否则

返回类型:

bool

示例

>>> is_envelope(ok(data="test"))
True
>>> is_envelope({"success": True})
False  # 缺少必需字段
>>> is_envelope("not a dict")
False
