Hook 系统 API¶

Hook 类型定义模块

提供 Hook 系统的类型定义，避免循环导入。

class plugin.sdk.hooks.HookMeta[源代码]¶

基类：object

Hook 元数据

target¶

Hook 目标 - 插件内: "entry_id" 或 "*" (所有 entry) - 跨插件: "plugin_id.entry_id"

Type:

str

timing¶

执行时机 - "before": 在目标 entry 执行前 - "after": 在目标 entry 执行后 - "around": 包裹目标 entry（可以控制是否执行） - "replace": 替换目标 entry

Type:

Literal['before', 'after', 'around', 'replace']

priority¶

优先级（越大越先执行）

Type:

int

condition¶

条件函数（返回 True 才执行 Hook）

Type:

str | None

target: str¶
timing: Literal['before', 'after', 'around', 'replace'] = 'before'¶
priority: int = 0¶
condition: str | None = None¶
property is_cross_plugin: bool¶

是否是跨插件 Hook

property target_plugin: str | None¶

获取目标插件 ID（跨插件时）

property target_entry: str¶

获取目标 entry ID
